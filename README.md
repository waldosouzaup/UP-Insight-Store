# UP InsightStore

Aplicativo analítico para pequenas lojas físicas, integrando Supabase e Google AI Studio para gerar dashboards, insights automáticos e respostas inteligentes sobre vendas, estoque e produtos.
Este repositório serve como portfólio e demonstração de um fluxo completo de análise aplicada ao varejo.

App in AI Studio: up-insight-store.vercel.app

Modelo de Planilha usada no projeto: https://drive.google.com/file/d/1c-3gOA3U5wJcBjoAgW6WoQyNt1B55LnH/view?usp=sharing

## 1. Visão Geral

O UP InsightStore foi desenvolvido para apoiar pequenos comerciantes na interpretação rápida e precisa dos dados do negócio.
Combinando um banco de dados em tempo real (Supabase) e uma camada de inteligência analítica (Google AI Studio), o app oferece:

Dashboards instantâneos

Previsões de vendas

Análise de giro de estoque

Identificação de produtos parados

Insights acionáveis para operação e marketing

IA conversacional para perguntas em linguagem natural

O foco é simplicidade, velocidade e aplicabilidade prática.

## 2. Funcionalidades Principais
Dashboards Automáticos

Faturamento diário, semanal e mensal

Ticket médio

Horários e dias de maior movimento

Produtos mais vendidos

Margem por categoria

## Análise Inteligente

Previsão de vendas (curto prazo)
  
Detecção de quedas bruscas de desempenho

Alerta de ruptura e estoque crítico

Produtos com baixa rotatividade

Recomendações de promoção com base na margem e no giro

## IA Conversacional

Permite que o usuário pergunte:

“Quais produtos mais venderam este mês?”

“Qual minha previsão de vendas para os próximos 7 dias?”

“Quais itens estão abaixo do ponto de reposição?”

“Qual categoria gera mais margem para a loja?”

## Integração com Supabase

Leitura em tempo real das tabelas:

vendas

estoque

produtos

APIs expostas via funções externas no Google AI Studio

Uso da chave pública (anon key), garantindo segurança do acesso
